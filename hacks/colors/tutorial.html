Welcome to the tutorial!

This tutorial will show you how to change the game colors using its configuration variables. You'll learn a few different ways to express colors on the web, and be able to customize the game to your liking.

Changeable colors

There are five colors you can control in the game:

  1. the background color (line 52)
  2. the player color while alive (line 59)
  3. the player color when dead (line 60)
  4. the robot color when alive (line 69)
  5. the robot color when dead (line 70)

The values you see within quotes at those locations are colors, even though they may not look like it. Let's try something simple to start...

On line 52, change "#002b36" to be "midnightblue".
